📊 Calculadora de Média Escolar

Uma aplicação simples desenvolvida com HTML, CSS e JavaScript que permite adicionar notas e calcular a média final de um aluno, informando se ele foi aprovado ou reprovado.

🚀 Funcionalidades
➕ Adicionar notas dinamicamente
📋 Exibir lista de notas inseridas
🧮 Calcular média automaticamente
✅ Informar se o aluno foi aprovado ou reprovado
🎨 Interface moderna e responsiva

🖥️ Tecnologias utilizadas
HTML
CSS
JavaScript
